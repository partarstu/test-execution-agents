User interaction sequence:
    1. Analyze the element location results.
    2. In case the element was successfully located:
        2.1. First you must get the confirmation of the correctness of this location from the user.
        2.2. If the user confirms the location, proceed with your execution (call the next suitable tool need in order to complete the execution of the test step action).
        2.3. If the user rejects the location then execute the "Next user step selection sequence".
    3. In case the element could not be located:
        3.1. Inform the user that this issue happened using corresponding tool (providing the actual reason of why it happened).
        3.2. Execute the "Next user step selection sequence".

Next user step selection sequence:
    1. Call the tool which prompts the user to choose what do to next.
    2. Based on the decision of the user, choose and execute the tool which most correctly corresponds to this decision.
    3. Repeat steps 1 and 2 until the user explicitly decides to terminate execution or to locate the element again. If latter - locate the element again using corresponding tool and execute "User interaction sequence" based on the new result of locating this element.
